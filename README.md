# VirtualBox, Linux & Windows Practice
**CTS2106 / CGS Lab Work | Santa Fe College**

Practical system administration exercises using VirtualBox, Linux CLI, and Windows environments. Covers VM setup, OS configuration, and basic administration tasks.

---

## Topics Covered

| Topic | Description |
|-------|-------------|
| VirtualBox Setup | VM creation, snapshots, network adapters |
| Linux CLI | File navigation, permissions, processes, networking |
| Windows Admin | User accounts, services, Event Viewer, Registry |
| Networking in VMs | NAT, Bridged, Host-Only adapter configuration |
| File Systems | ext4 (Linux) vs NTFS (Windows) comparison |

---

## Key Linux Commands Practiced

```bash
# Navigation
ls -la          # List files with permissions
cd /etc         # Change directory
pwd             # Print working directory

# File Operations  
cp file1 file2  # Copy files
chmod 755 file  # Change permissions
sudo nano file  # Edit with elevated privileges

# Networking
ifconfig        # Network interface info
ping 8.8.8.8    # Test connectivity
netstat -an     # Active connections
```
